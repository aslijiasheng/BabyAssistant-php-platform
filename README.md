## Introduction

*BabyAssistant* is a Early Learning Project.

It is not a complete project, need rely node-server running.

## Rely

* memcached
* elasticsearch
* yii2-advanced
* yii-elasticsearch
* [BabyAssistant-node](https://github.com/aslijiasheng/BabyAssistant-node)
* php5.5 or lastest
